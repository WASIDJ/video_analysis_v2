## 指标分类体系

系统将生物力学指标分为 **9 个大类**，基于解剖学和运动学原理：

| 分类 | 英文 | 描述 | 指标数量 |
|------|------|------|----------|
| 关节角度（矢状面） | `joint_angle_sagittal` | 矢状面关节角度 | 6 |
| 关节角度（冠状面） | `joint_angle_frontal` | 冠状面关节角度 | 4 |
| 关节角度（水平面） | `joint_angle_transverse` | 水平面关节角度 | 1 |
| 位置关系 | `position` | 关键点相对位置 | 2 |
| 位置对齐 | `position_alignment` | 身体部位对齐度 | 2 |
| 位置对称 | `position_symmetry` | 左右对称性 | 1 |
| 活动范围 | `rom` | 关节活动范围 | 1 |
| 体块轮廓 | `segment` | 身体段轮廓特征 | 3 |
| 稳定性 | `stability` | 动作稳定性指标 | 1 |

## 指标计算原理

### 角度计算
```python
# 关节角度计算（余弦定理）
def calculate_angle(a, b, c):
    # a, b, c 为三个关键点的坐标
    ba = a - b
    bc = c - b
    cosine_angle = np.dot(ba, bc) / (np.linalg.norm(ba) * np.linalg.norm(bc))
    angle = np.arccos(cosine_angle)
    return np.degrees(angle)
```

### 距离计算
```python
# 归一化距离计算
def normalized_distance(p1, p2, reference_length):
    # 计算两点欧氏距离，用参考长度（如肩宽）归一化
    raw_distance = np.linalg.norm(p1 - p2)
    return raw_distance / reference_length
```

### 曲率计算
```python
# 脊柱曲率计算
def calculate_curvature(points):
    # points: 脊柱关键点序列 [p1, p2, p3, ...]
    # 计算相邻线段夹角变化率
    angles = []
    for i in range(1, len(points)-1):
        angle = calculate_angle(points[i-1], points[i], points[i+1])
        angles.append(angle)
    return np.mean(angles)  # 平均曲率
```

## 指标组合与衍生

虽然基础指标只有 21 个，但通过组合可以衍生出 **50+ 种分析维度**：

### 1. 时间维度衍生
- **峰值角度**：动作过程中的最大角度
- **谷值角度**：动作过程中的最小角度
- **角度变化率**：角度随时间的变化速度
- **角度稳定性**：角度的标准差/变异系数

### 2. 空间维度衍生
- **左右差异**：左侧 vs 右侧的对比
- **前后对比**：起始位置 vs 结束位置
- **上下对比**：上肢 vs 下肢的协调性

### 3. 动作阶段衍生
- **起始角度**：动作开始时的角度
- **中间角度**：动作执行中的角度
- **结束角度**：动作完成时的角度
- **转折角度**：动作方向改变时的角度

### 4. 复合指标衍生
- **关节协调性**：多个关节角度的同步性
- **身体对称性**：左右侧指标的对称程度
- **动作流畅度**：角度变化的平滑程度
- **能量效率**：角度变化与做功的关系

## 生物力学意义总结

### 1. 损伤预防指标
- **`knee_valgus`**：ACL损伤风险 → 值越大风险越高
- **`lumbar_curvature`**：腰椎损伤风险 → 值越大风险越高
- **`trunk_lean`**：下背部压力 → 值越大压力越大

### 2. 动作质量指标
- **`knee_flexion`**：动作幅度 → 值适中为佳
- **`hip_flexion`**：髋关节参与度 → 值适中为佳
- **`ankle_dorsiflexion`**：踝关节灵活性 → 值适中为佳

### 3. 对称性指标
- **`knee_symmetry`**：双侧对称性 → 值越小越对称
- **`pelvic_obliquity`**：骨盆平衡 → 值越小越平衡

### 4. 代偿模式指标
- **`trunk_lateral_flexion`**：躯干代偿 → 值越小代偿越少
- **`knee_flexion_compensation`**：膝关节代偿 → 值越小代偿越少
- **`shoulder_lift_ratio`**：耸肩代偿 → 值越小代偿越少

## 指标选择策略

### 1. 动作特异性选择
- **下肢主导动作**（深蹲、弓步）：优先选择 `knee_flexion`, `knee_valgus`, `hip_flexion`
- **上肢主导动作**（推举、划船）：优先选择 `shoulder_lift_ratio`, `trunk_lean`
- **核心主导动作**（平板、侧平板）：优先选择 `lumbar_curvature`, `pelvic_tilt`

### 2. 错误模式针对性选择
- **膝关节问题**：重点监控 `knee_valgus`, `knee_symmetry`
- **下背部问题**：重点监控 `lumbar_curvature`, `trunk_lean`
- **不对称问题**：重点监控 `pelvic_obliquity`, `knee_symmetry`

### 3. 训练目标导向选择
- **力量训练**：关注 `knee_flexion`（深度）, `trunk_lean`（稳定性）
- **康复训练**：关注 `knee_valgus`（稳定性）, `hip_abduction`（功能）
- **姿势矫正**：关注 `lumbar_curvature`, `thoracic_curvature`, `pelvic_tilt`
